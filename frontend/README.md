## Frontend – AI Copilot

This is the frontend of the AI Copilot project, built with **Vue 3**, **Vite**, **Pinia**, **Vue Router**, and **Tailwind CSS**.

### 🚀 Getting Started

```bash
cd frontend
npm install
npm run dev
```
### 🗂️ Project Structure

- `src/components/` – Reusable UI components  
- `src/views/` – Page-level views (e.g., Login, Dashboard)  
- `src/store/` – Global state management (Pinia)  
- `src/router/` – Application routing
